# PRODIGY_GA_03

## Markov Chain Text Generator

This project generates text using a Markov Chain model.

The model learns word transitions from a training text and predicts the next word based on probability.

## Technologies Used

- Python
- Gradio
- Random Module

## Features

- Custom Training Text Input
- Start Word Selection
- Markov Dictionary Generation
- Text Generation using Markov Chain

## Workflow

Training Text
↓
Markov Dictionary
↓
Start Word
↓
Generated Text

## Example

Training Text:

I love AI and AI loves learning

Start Word:

I

Generated Text:

I love AI and AI loves learning
